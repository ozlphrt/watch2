# Watch Physics Simulation

3D watch face with physics simulation - cubes falling onto watch face with interactive arms.

## Setup

```bash
npm install
npm run dev
```

## Coordinate System

- **Handedness**: Right-handed
- **Up axis**: Y
- **X**: East(+)/West(-)
- **Y**: Up(+)/Down(-)
- **Z**: South(+)/North(-)
- **Origin**: Center of world grid
